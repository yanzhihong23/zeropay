1、注册并登录

通过以下地址注册后登录

https://account.chsi.com.cn/account/preregister.action?from=archive-login

点击 【立即登录】

2、学信档案

点击 【申请该学籍在线验证报告】

点击 【在线支付】

点击 【确认】后，并用支付宝支付2元费用

3、获得查询码

返回原界面，点击【已完成支付】

获得查询码（位置如下）

注册学信网帐号：
https://account.chsi.com.cn/account/preregister.action?from=archive-login

已有学信网账号，请登录： https://account.chsi.com.cn/passport/login
